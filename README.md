# Ilsie Studio | Interactive Portfolio 💽

**Ilsie Studio** adalah sebuah website portofolio interaktif *single-page* yang mengemas informasi profil pribadi, rekam jejak akademis, serta pencapaian organisasi ke dalam estetika studio musik retro-futuristik ala tahun 2000-an (Y2K). Website ini dirancang sebagai ruang eksplorasi digital yang kreatif, berkarakter, namun tetap terstruktur secara informasi.

Proyek ini menerapkan beberapa hukum pengalaman pengguna (*UX Laws*) seperti *Miller's Law* untuk pengelompokan informasi, *Jakob's Law* untuk navigasi, dan *Law of Similarity* untuk konsistensi komponen tombol, guna memberikan pengalaman penjelajahan yang berkesan bagi para pengunjung.
---
## 🚀 Cara Instalasi

Proyek ini dibangun menggunakan teknologi web murni (*vanilla web technologies*) sehingga tidak memerlukan proses instalasi *dependency* yang rumit. Berikut adalah langkah-langkah untuk menyiapkan proyek ini di perangkat lokal Anda:

1. Salin (*clone*) repositori ini ke direktori lokal komputer Anda:
```bash
   git clone [https://github.com/fildzahalfitri/ilsie-studio.git]
Buka folder proyek ilsie-studio menggunakan aplikasi code editor pilihan Anda (direkomendasikan menggunakan Visual Studio Code).

Pastikan struktur file utama seperti index.html, style.css, dan script.js berada dalam satu direktori kerja yang sama.

💻 Penggunaan
Untuk menjalankan dan melihat hasil enkapsulasi kode dari website ini secara langsung di browser lokal, ikuti panduan berikut:
1. Pastikan Anda telah menginstal ekstensi Live Server oleh Ritwick Dey di Visual Studio Code.
2. Buka file index.html melalui editor.
3. Klik tombol "Go Live" pada status bar di pojok kanan bawah editor, atau klik kanan pada area kode index.html lalu pilih opsi "Open with Live Server".
4. Aplikasi akan otomatis berjalan pada server lokal (biasanya pada alamat http://127.0.0.1:5500/) dan membuka peramban (browser) bawaan Anda.
5. Anda dapat mulai berinteraksi dengan mengklik ikon pada layar laptop di fitur The Control Room atau menggeser komponen carousel pada bagian rekam jejak.

🌐 Akses Online: Website ini juga sudah dideploy secara live dan dapat diakses kapan saja secara responsif melalui berbagai perangkat di tautan berikut: ilsie-studio.vercel.app

🤝 Kontribusi
Proyek ini awalnya dikembangkan sebagai proyek tugas akhir akademis untuk merepresentasikan identitas profesional penulis. Meskipun demikian, masukan, pelaporan kutu (bug), atau saran pengembangan visual sangat terbuka bagi siapapun.

Jika Anda ingin berkontribusi untuk mengembangkan fitur baru:
1. Lakukan Fork pada repositori ini.
2. Buat branch fitur baru Anda (git checkout -b fitur-baru-keren).
3. Lakukan dokumentasi perubahan melalui commit (git commit -m 'Menambahkan fitur baru X').
4. Unggah perubahan tersebut ke branch Anda (git push origin fitur-baru-keren).
5. Buat sebuah Pull Request agar dapat ditinjau lebih lanjut.
